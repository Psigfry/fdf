# fdf

FdF - это средство просмотра 3D-карт. (Поддерживается только macOS)

## Installation
Clone the repository to a directory.</br>
```
https://github.com/Psigfry/fdf.git
```
</br>
Run make to compile the library.</br>
```
cd fdf && make && make clean
```
</br>
